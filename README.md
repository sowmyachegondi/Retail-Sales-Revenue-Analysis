# Retail Sales & Revenue Analysis

## Overview
This project analyzes retail sales data using Python to identify sales trends, top products, valuable customers, and revenue patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Top 10 Best-Selling Products
- Top 10 Customers by Revenue
- Monthly Sales Trends
- Country-wise Revenue Analysis
- Pareto Analysis (80/20 Rule)

## Dataset
Online Retail II Dataset from Kaggle

## Project File
- Retail_Sales_Revenue_Analysis.ipynb

## Key Insights
- Identified best-selling products
- Found top revenue-generating customers
- Analyzed seasonal sales trends
- Compared revenue across countries
- Applied Pareto Principle for customer analysis
